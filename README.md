# Mental Health DBMS System

## Overview
A relational database system that analyzes user responses to mental health questionnaires and classifies them using score ranges.

## Features
- Normalized database schema
- Trigger-based score calculation
- SQL queries demonstrating DBMS concepts
- Python-MySQL integration

## Folder Structure
- database/ → SQL scripts
- src/ → Python code
- docs/ → Report
- results/ → Outputs

## How to Run
1. Run schema.sql
2. Run insert_data.sql
3. Run queries
4. Run app.py
